## ⚠️ DEMO GOTCHAS & FIXES

### **If Dark Mode Doesn't Work:**
- Refresh page (Ctrl+F5)
- Clear cache
- Check localStorage in DevTools

### **If FAB Doesn't Appear:**
- Scroll down (it's fixed at bottom right)
- Check browser zoom (should be 100%)
- Try different browser

### **If Trends Don't Show:**
- Make sure data was logged previously
- Check backend is running
- Try refreshing dashboard

### **If Badges Don't Unlock:**
- Explain they need historical data
- Show what conditions unlock each
- Point to progress bars

### **If Something Breaks:**
- Have backup demo account ready
- Show code in IDE
- Explain technical architecture
- Pivot to strengths
